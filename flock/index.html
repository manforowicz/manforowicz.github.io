Boid Flocking Algorithm

Flocking

Ever look up at the sky, and see an amazing flock of birds? How do they manage to navigate harmoniously without collisions?

To better understand their behavior, I created a boids simulation. Boids are an artificial life algorithm created by Craig Reynolds in 1986.

I wrote this project in Rust using the Macroquad game engine, and compiled it to web assembly (wasm). View this website's code on Github.

Sliders

Mess around with the following sliders to discover new behaviors.

Slider Function
Population Set how many boids are simulated.
Spacing goal The distance every boid tries to maintain from its neighbors.
Seperation strength How strongly the boids try to return to spacing goal if they're too close.
Cohesion strength How strongly the boids try to return to spacing goal if they're too far.
Alignment Strength What absolute speed the boid attempts to maintain.
Target Speed What absolute speed the boid attempts to maintain.
Speed Strength The strength at which the boid accelerates or deccelerates to maintain target speed.
